OpenCache Project

overview // docs // tutorial // research // about

OpenCache is an open-source distributed content caching platform. Currently in development, OpenCache's operation is transparent to requesting clients. Through the use of OpenFlow, requests are redirected dynamically and on-the-fly. In addition, the OpenCache Project is working towards a well defined and open interface for the management and administration of cache deployments, details of which can be found in our documentation.

Source code is hosted on github. To get started, please follow our tutorial. For examples of our recent scientific publications, please see our research section. More information on the team behind OpenCache can be found in the about section.
